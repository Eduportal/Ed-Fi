Welcome to the Ed-Fi Operational Data Store API

The Ed-Fi ODS Web API allows applications to read and write education data stored in an Ed-Fi ODS through a secure REST interface. The Ed-Fi ODS API supports both transactional and bulk modes of opteration.

Use the following steps to explore the technical preview of the Ed-Fi ODS API:

  1. Populate Sandbox Administrative Database with Users
    Create test users and an administrative user in order to execute tests and access the Sandbox Administrative Portal (see step two).
  2. Create an Ed-Fi ODS API Sandbox
    To access the data in the Ed-Fi ODS database using the Ed-Fi ODS API, you need to provision an API sandbox using the API Sandbox Administration Portal. Login with a user name of "test@edfi.org" and a password of "***REMOVED***". We recommend that you change your password as soon as you log in.
  3. Review the Ed-Fi ODS API Documentation
    Review the Ed-Fi ODS Documentation Website in order to understand the methods for interacting with data in your Ed-Fi ODS API sandbox.
